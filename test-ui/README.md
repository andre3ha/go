# Test UI for LR2

Эта папка содержит простую HTML-страницу для теста API на другом порту.

## Запуск

Из корня проекта:

```powershell
cd test-ui
python -m http.server 5500
```

После этого открой:

- `http://localhost:5500`

По умолчанию страница отправляет запросы на:

- `http://localhost:8080`

Если backend работает на другом адресе/порту, поменяй `Base URL API` на странице.
